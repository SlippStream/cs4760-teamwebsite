The Critters
Group 6 Code Critiquer
Code critiquer is a static analysis tool for finding common anti-patterns in user-submitted code snippets. Similar to grammar or plagiarism checkers in which users can submit either their full source code or erroring segments, and have the app respond with feedback pertaining to the found anti-pattern (bugs, errors, etc.) found within the given code. Currently, Code Critiquer will be used on Java code with the goal of implementing MatLab support, if time permits.
